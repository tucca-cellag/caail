# TUCCA / Tufts lockup assets

The official **Tufts University — Center for Cellular Agriculture** horizontal lockups, from the
Tufts brand kit. Used on the CAAIL site to attribute the project to TUCCA (hero + footer).

- `cca-horiz-blkbl.png` — black + Tufts-blue lockup, for light backgrounds.
- `cca-horiz-white.png` — all-white lockup, for dark backgrounds (dark mode).

Both are the 2× RGB PNGs (1093×169) from the official kit, rendered by `TuccaLockup.astro`, which
swaps between them on `html[data-theme='dark']`.
